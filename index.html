Projeto desenvolvido no WordPress com HTML e CSS personalizados.

💻 Sobre o Projeto

O Mapa das Estrelas é um projeto desenvolvido em WordPress com personalizações em HTML e CSS. O objetivo foi criar páginas organizadas, responsivas e visualmente atrativas, melhorando a experiência do usuário e a navegação pelo conteúdo.
